JQuery动态修改a标签链接地址

发表于 2017-02-10 更新于 2020-08-19 分类于 JAVAScript ， 编程 阅读次数：
本文字数： 594 阅读时长 ≈ 1 分钟

在一个项目中，想隐藏真实的链接地址，使用JQuery的事件可以解决这个问题，代码如下：

$(document).ready(function(){
//动态修改href值
$(“#aid”).mouseenter(function(){//鼠标划过或停留时显示正常地址，并保证点击后仍然显示正常地址
$(this).attr(‘href’,’http://www.baidu.com/');
}).click(function(){//链接被点击时打开实际地址
$(this).attr(‘href’,’http://126.am/Ho9PX2');
});
});

代码执行的结果是，普通浏览者点击后会打开我们想要的地址，而不是显示的那个地址。如果同样的动作要执行多次，也可以封装成一个JQuery函数或插件，

$.fn.ChangeHref = function(show_url, click_url) {
$(this).mouseenter(function(){
$(this).attr(‘href’,show_url);
}).click(function(){
$(this).attr(‘href’,click_url);
});
};

$(“#aid”).ChangeHref(‘http://www.baidu.com/, ‘http://126.am/Ho9PX2');//调用函数
